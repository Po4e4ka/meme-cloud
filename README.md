# MemeCloud

MemeCloud — это личное хранилище мемов. Пользователь входит в аккаунт, загружает изображения или короткие видео с телефона или компьютера и организует их с помощью тегов для быстрого поиска по галерее. Все файлы хранятся в приватной папке пользователя в бакете.

## Основные возможности
- Личные аккаунты с приватными галереями.
- Загрузка изображений и видео с веба или мобильных устройств.
- Теги и поиск для быстрого доступа к мемам.
